## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

--Node v17.9.0
--Npm v8.5.1
--Composer v2.0.8
--Text editor (VsCode)

### Install

1. Clone the repo
   ```sh
   git clone git@github.com:Semi-dios/landing-page-google.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
   ```sh
   yarn install
   ```
